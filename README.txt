# Pizza Azura - Production Package

## Quick Setup:
1. Upload semua file ke public_html/ di cPanel
2. Set permissions: application/cache/ dan application/logs/ -> 777
3. Edit file HTML, tambahkan: <script src="js/api-helper.js"></script>
4. Buka: https://pizzaazura.my.id

Baca UPLOAD_GUIDE_SHARED_HOSTING.md untuk detail lengkap!
